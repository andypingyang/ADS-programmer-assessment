# ADS Programmer Assessment

This repository contains solutions for an ADS programmer assessment, including:

## Structure
- **question_1_sdtm/**  
  SDTM DS domain creation using sdtm.oak.
- **question_2_adam/**  
  ADSL derivation using ADaM principles.
- **question_3_tlg/**  
  AE summary tables and visualizations.
- **clinical_data_agent-2.py**  
  A simple clinical trial data agent that translates natural language
  questions into AE dataset queries.

## How to run
Scripts are designed to run in Posit Cloud / RStudio using
pharmaverse and admiral example datasets.
